​ 第二范式：

多值依赖(MVD)

书上对多值依赖的定义：

def 多值依赖：

​ 设$U$是关系模式$R$的属性集，$X$和$Y$是$U$的子集，$Z=R-X-Y$，小写的$xyz$表示属性$XYZ$的值。对于$R$的关系$r$，在$r$中存在元组$(x,y_1,z_1)$和$(x,y_2,z_2)$时，也就存在元组$(x,y_2,z_1),(x,y_1,z_2)$，那么称多值依赖$X\rightarrow\rightarrow Y$在模式$R$上成立。

个人觉得这个定义……非常的丑……

关于MVD的推理规则集

M1(MVD的补规则)：若$X\rightarrow\rightarrow Y$,则$X\rightarrow\rightarrow (U-XY)$

M2(MVD的增广性)：若$X\rightarrow\rightarrow Y, V \subseteq W \subseteq U$，则$WX\rightarrow\rightarrow VY$

M3(MVD的传递性)：若$X\rightarrow\rightarrow Y, Y\rightarrow\rightarrow Z$，则$X \rightarrow\rightarrow (Z-Y)$
