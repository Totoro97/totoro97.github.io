来自大三的 hello world

发表于 2017-09-23

我也不知道为什么会取这个标题……可能只是因为我现在大三了吧……

我嘛，总是会过高地去期望自己……是不是慢慢地到天花板了。

走进了一个循环吧。想来2015到2017的轨迹，和2012到2014的轨迹相差无几。信心满满到逐渐地放弃治疗。最终还是干不过自己的缺点和现实。

今天只不过是很正常的大三的一天。当然可能每天都只不过是这样。只不过是今天练习赛打崩了而已。只不过、只不过、只不过。能够颓那么久也是醉了。

果然还是要靠做事情去填充愧疚感以及一些其他的各种负面情绪对吧。有时候是真的讨厌自己的这种明明可以去做事情还是颓的状态啊。

可能我最开心的时刻已经停留在小学了(2333333333333)。真的很羡慕小学时的我的各种狂拽炫酷(其实也不)不怕得罪其他人的状态(233333333333)。不过讲道理我小学的时候很容易哭鼻子，哇我竟然想起了一些小学时的黑历史，真是有趣，不过也可以理解为年少无知吧，所以会很任性，自我中心啥的。

不过啊，小学时自我中心，现在就不自我中心了么？不哇，一直都很自我中心啊，现在也是，只不过是外在的表现形式变了而已。(写到这里，潜意识里就会期望能有个人会对我说：“不啊，你没有很自我中心啊，你人很好的。”，没错啊，就是这么虚伪，这种降低自己评价以乞求外来的相对好听的评价，不就是我经常干的事情么。)

等等我好像写偏了。其实写这些东西也是自我中心的表现吧。

你总得承认，有些高度是你达不到的，所以还是不要抱着些虚虚的期望了。

真切地感受到，大学里的大部分时光我是不开心的。当然说不开心不是“伤心”或“难过”，就是单纯的压抑吧。

所以说为什么从大一开始就一直都有健身的习惯呢因为这可能是最适合一个人做的而且能够稍微释放自己的运动0.0。所以心情不好，就去健身咯。

心里抵触争锋相对，所以很少吵架，得到的是半天或一整天的自我消化。当然如果有人突然对我说“哇你好惨啊好可怜啊”我可能又得花上半天或一天的时间去消化这小小的善意……所以我真的真的很讨厌我这种性格。

真的。

花上这么多时间去消化负面情绪，却没有去想想现在真正应该做什么事情。但是又有谁能说清楚“真正应该做的事情”到底是什么。

结果导向可能是个不错的方法。做啥都好，想想结果，也比消化要好。得想办法慢慢地去减消化时间。

只是少了提醒而已。

还是得天天提醒，否则就会做梦啊。

审视啥的。

你他妈就不能别这么敏感么，想想办法啊，向妈妈学习啊，啊，啊，啊。
