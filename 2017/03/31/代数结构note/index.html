代数结构

代数系统

代数系统：载集，运算，n元运算，运算封闭性

单位元：

  • 左单位元，右单位元； 逆元 = 左单位元 & 右单位元
  • 单位元若存在则唯一。

逆元：

  • 左逆元，右逆元(对于一个$a \in S$)，逆元 = 左逆元 & 右逆元
  • 当运算满足结合律且a有逆元时，逆元是唯一的。

零元：

  • 左零元，右零元，零元。

分配律：

  • 左分配律：$a (b + c) = a b + a c$ (称关于+有左分配律) (右类似)

代数系统的同态：

​ 同态映射：运算相容

​ $\phi(x) : S \rightarrow T$

​ 满同态映射得到两个系统同态。

​ 同态的性质的传递$S \rightarrow T​$：交换律，单位元

代数系统的同构：

​ 当$\phi(x)$是双射时，得到同构$S \cong T$，可看成是同一个代数系统。

商系统：

​ 在代数系统$[s, *]$上新定义一个相容等价关系$t$。

​ 称$[s/t, \bigoplus]$是一个商系统

群

基本概念：

​ 半群：满足结合律的代数系统

​ 拟群：含单位元的半群

​ 群：每个元素都有逆元的拟群

​ 阿贝尔群：交换群

​ 群的阶：$|G|$，有限群，无限群

定理13.5：群G满足消去律。

定理13.6：半群G是群，当且仅当下列条件满足

​ 1.对任意$g \subseteq G$ 存在左单位元(存在$e’g=g$)

​ 2.对任意$g \subseteq G$ 存在左逆元(存在$g’g = e’$)

​ 证法：先证明左逆元就是逆元，再证明左单位元就是单位元。

定理13.7 : 半群G是群，当且仅当，对任意的$a,b \in G$必存在$x,y \in G$，使得$ax=b, ya=b$

定理13.8 : G为有限半群，它是群，当且仅当运算满足消去率。

克莱因四元群

变换群，置换群 : $|S| < +\infty$下的变换群就是置换群

n次对称群

n次交代群：所有偶置换构成的集合

循环群：有生成元

​ 无限循环群同构于加法循环群

​ 有限循环群同构与同阶同余类加法循环群

子群：子集+群

判断子群的三个定理(满足任意一个条件即可)：

  • 封闭+任意元素均有逆元
  • $\forall a,b \in H, ab^{-1} \in H$
  • 有限+封闭

陪集：

​ 设H为G的子群，取G中一个固定元素g，用g与H中的每个元素进行乘法运算，将其结果组成一个集合，记为g，称其为H的左陪集，同理有右陪集。

引理13.1 : 对于有限子群H有 $|H| = |gH| = |Hg|$

引理13.2 : H为G的子群，只可能有$Hg_1=Hg_2$或$Hg_1 \cap Hg_2 = \emptyset$

推论13.7 : 子群$H$对$G$的所有右陪集构成对集合$G$的一个划分

定理13.17 : $|H|\times k =|G|$

H在G中的指数k，即拉格朗日定理

推论13.8 : 当$|G|\ is\ prim$，$G$是循环群，很好证。

正规子群：对任意的g，左陪集=右陪集那么H就是正规子群(不变子群)

定理13.18 : (正规子群的等价定义)：$\forall g \in G, h \in H \rightarrow g^{-1}hg\in H$

正规子群下可以得到商群:$G/H$ , 所有不同的陪集组成的集合

引理13.3 : 运算结果与代表元的选取无关

证明时充分利用正规子群的性质。

引理 : $[G/H; \odot]$是群。

群的同态与同态基本定理

Cayley定理：任意有限群必同构于一个同阶的置换群

证明时，构造一个映射(这个映射是元素到变换的映射)

同态核：使得$\phi(x)=e’$的$x$的集合

同态核K是G的正规子群

定理13.19 : 群G同态于它的任一商群，这种同态称为自然同态

定理13.20 : $G/K \cong \phi(G)$ ，$\phi$是同态映射

环

环：代数系统$[R;+,]$，$[R;+]$是阿贝尔群，$[R;]$是半群，满足分配律。

一般定义0是+下的单位元，1是*下的单位元(如果单位元存在的话)

定理14.1 : 在环中有

​ 1.$a0 = 0a = 0$

​ 2.$a(-b) = (-a) b = -(a * b)$

加法的单位元经常被叫做环的“零元”。

定义14.2 : 有单位元环

定义14.3 : 零因子

整环：*满足交换律，且没有零因子。

定理14.2 : 整环满足消去律

除环：在*运算下有单位元，且每个非零元素有逆元

域：乘法满足交换律的除环

子环与环同态

定义14.6 : 子环：子集＋环

定理14.4 : 环的中心$C={x|x \in R,\forall a \in R, ax = xa}$是$R$下的子环

单位子环：$E = {ne| n \in Z}$

使$ke=0$的最小正整数称为环$R$的特征数，记为$char\ R$

定理14.5 : 令$p = char\ R$，$1 \in R$，有

​ 1.$\forall a \in R, pa = 0$，当R是整环时，p也是使任意的a，pa=0的最小正整数

​ 2.当R是整环时，$char\ R$为质数或0。

环同态：

​ 满同态映射，满足运算相容就可以。

定理14.6 : 对于同态映射$\phi$：

​ 1.$\phi(0) = 0’$

​ 2.如果$R,R’$均为有单位元环，当$\phi$是满射或者($R’$无零因子环且不是零同态)，$\phi(e) = e’$

​ 3.$\phi(R)$是$R’$的子环

定理14.7 : 若$Char\ R \ne 0$，则$\phi(a) = a^{Char\ R}$是同态映射而且是单射。

重要结论 : 设$Char\ R = p$，有$(a + b)^p = a^p + b^p$ , $(a-b)^p = a^p - b^p$

有限域

定理15.12：$F$为有限域，则存在素数$p$，自然数$m \geq 1$，使得$|F| = p^m$

def 迦罗瓦域：$p^m$阶迦罗瓦域，记为$GF(P^m)$。

定理15.13：设$char F = p​$，$\Delta​$为其所包含素域，$|F|=p^m​$，则$F​$是$x^{p^m}-x​$在$\Delta​$上的根域。

推论15.5 $GF(p^m)$中任一元素在其素域$\Delta$上均有一个极小多项式。

定理15.14：任两个同阶的迦罗瓦域必同构。

推论15.16：$GF(p^m)$中的元素恰为多项式$x^{p^m}-x \in Z_p[x]$的$p^m$个根。
